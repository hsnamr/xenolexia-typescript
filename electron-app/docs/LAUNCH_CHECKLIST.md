## App Store (iOS)

### Developer Account
- [ ] Apple Developer Program membership active
- [ ] App ID registered (com.xenolexia.app)
- [ ] Certificates and profiles created
- [ ] App Store Connect app record created

### App Information
- [ ] App name confirmed available
- [ ] Primary category: Education
- [ ] Age rating: 4+
- [ ] Content rights declarations

### Store Listing
- [ ] App icon uploaded (1024x1024)
- [ ] Screenshots uploaded
  - [ ] 6.5" iPhone (1284 x 2778)
  - [ ] 5.5" iPhone (1242 x 2208)
  - [ ] 12.9" iPad (2048 x 2732)
- [ ] App preview video (optional)
- [ ] Description written (4000 chars)
- [ ] Keywords set (100 chars)
- [ ] Support URL
- [ ] Marketing URL
- [ ] Privacy Policy URL

### Build & Submit
- [ ] Archive build created
- [ ] Build uploaded to App Store Connect
- [ ] Build selected for submission
- [ ] Export compliance (HTTPS: Yes, Encryption: None)
- [ ] Advertising identifier: No
- [ ] Review notes added

### Review Preparation
- [ ] Demo instructions written
- [ ] Test account (if needed): N/A
- [ ] Contact information current

## Play Store (Android)

### Developer Account
- [ ] Google Play Developer account active
- [ ] App created in Play Console
- [ ] Package name: com.xenolexia.app

### App Content
- [ ] Privacy policy URL
- [ ] App access: No special access
- [ ] Ads declaration: No ads
- [ ] Content rating questionnaire completed
- [ ] Target audience: Not designed for children
- [ ] News app: No

### Store Listing
- [ ] App icon (512 x 512)
- [ ] Feature graphic (1024 x 500)
- [ ] Screenshots
  - [ ] Phone (at least 2)
  - [ ] 7" tablet (at least 1)

- [ ] App category: Education

### Data Safety
